Como usar git localmente
1. git clone URL
2. cd directorio
3. git status/commit
4. gedit README.md y modificado
5. git status
6. git diff
--------------------------------------------------------------------
7. git add README.md #Modificamos el fichero
8. git commit README.md --no-verify
9. git push #Envia los cambios del repositorio local al de la nube
